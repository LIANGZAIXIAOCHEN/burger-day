# D:\workspace - 工作空间

## 目录结构
- /d/workspace - 主工作目录

## 使用说明
这是一个通用的工作空间，用于存放各类项目文件。

## 已安装的插件
- github - GitHub MCP 集成
- code-review - 自动化代码审查
- code-simplifier - 代码简化优化
- commit-commands - Git 工作流
- claude-md-management - CLAUDE.md 管理
- feature-dev - 功能开发工作流
- skill-creator - 创建自定义 Skills
- playwright - 浏览器自动化测试
- playground - HTML 交互式游乐场
- claude-code-setup - 代码库分析与自动化建议
- security-guidance - 安全审查
- context7 - 框架文档查询
- session-report - 会话报告
- frontend-design - UI/UX 前端设计
